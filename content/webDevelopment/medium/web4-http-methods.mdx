# HTTP Methods

Understanding HTTP methods is fundamental to RESTful API design.

PUT is idempotent and replaces the entire resource with the request payload. Multiple identical PUT requests have the same effect as a single request. PATCH is used for partial updates and is not necessarily idempotent.
