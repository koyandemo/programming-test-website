# Garbage Collection Algorithm

All modern Java garbage collectors (Serial, Parallel, G1, ZGC) use survivor spaces as part of generational garbage collection. The young generation is divided into Eden and two survivor spaces (S0 and S1) to reduce memory fragmentation and improve collection efficiency.
