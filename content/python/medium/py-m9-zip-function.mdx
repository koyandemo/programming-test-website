# Zip Function

Understanding how zip() combines iterables.

The `zip()` function combines multiple iterables element-wise, stopping when the shortest iterable is exhausted. Since [1, 2] has only 2 elements, zip stops after pairing the first 2 elements, resulting in [(1, 'a'), (2, 'b')].
