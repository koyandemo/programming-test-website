# List Comprehension with Condition

List comprehensions can include conditional filtering.

The expression `[x for x in range(10) if x % 2 == 0]` creates a list of even numbers from 0 to 9. The `if` clause filters out odd numbers, resulting in [0, 2, 4, 6, 8].
