# __getattr__ vs __getattribute__

`__getattribute__()` is called for every attribute access and can intercept all lookups. `__getattr__()` is only called as a fallback when the attribute isn't found through normal means. Overriding `__getattribute__()` requires care to avoid infinite recursion.
