title: "SQL Window Functions and Group By"
statement: "An SQL Window function requires the use of the `GROUP BY` clause to define the partition over which the function operates."
difficulty: "HARD"
category: "Database"
correctAnswer: false
explanation: "Window functions use the `PARTITION BY` sub-clause within the `OVER` clause to define the window, which is distinct from the aggregation performed by `GROUP BY`."
