# Database Indexing Strategy

This question tests advanced understanding of database indexing trade-offs.

While indexes speed up SELECT queries, they can slow down INSERT, UPDATE, and DELETE operations because the index must be updated with each modification. Tables with frequent write operations may suffer performance degradation from too many indexes.
