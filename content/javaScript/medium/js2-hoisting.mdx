# JavaScript Hoisting

This question tests your understanding of hoisting behavior with modern JavaScript variable declarations.

The Temporal Dead Zone (TDZ) is a behavior in JavaScript where variables declared with `let` and `const` cannot be accessed before their declaration, even though they are hoisted.
